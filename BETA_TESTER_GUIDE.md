# Defender™ Beta Tester Guide

## Start Safely

1. Extract the complete release ZIP before opening the app.
2. Run `Defender Beta.exe` from the extracted folder.
3. Use test or redacted documents for early testing whenever possible.
4. Remember that all generated tax PDFs are non-final review copies.

## What To Test

- App opens without a browser, account, or sync prompt.
- Sources can be added, reviewed, and added to the return.
- Questions can be answered and changed.
- Review PDFs export and the output folder opens.
- Closing the desktop window completely stops Defender Beta.

## Reporting A Problem

1. Leave the affected workflow in place when possible.
2. Open App Settings.
3. Choose `Export support snapshot` and confirm.
4. Send the snapshot with the steps that produced the problem.

The support snapshot is designed to exclude uploaded tax documents, generated PDFs, and obvious personal identifiers. Review the ZIP before sending if desired.

## Resetting Test Data

App Settings includes `Clear local test data`. This is destructive and requires confirmation. It removes local sources and workflow state while preserving app settings.
